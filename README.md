# React recipe app
live demo - https://data111a.github.io/React-recipe-app/
#

## Technologies used
-   [React](https://reactjs.org/)
- React Hooks
- Routing
- Styled components
- API


# Setup
- Clone repo
```
$ git clone https://github.com/data111a/React-recipe-app
```
- Install dependencies
```
$ npm install
```
- run on local machine
```
$ npm run dev
```
